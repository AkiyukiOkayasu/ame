◆ ts_cmsis_cv()

void ts_cmsis_cv ( void  )

This is CORE Validation test suite.

Program flow:

  1. Test report statistics is initialized
  2. Test report headers are written to the standard output
  3. All defined test cases are executed:
    • Test case statistics is initialized
    • Test case report header is written to the standard output
    • Test case is executed
    • Test case results are written to the standard output
    • Test case report footer is written to the standard output
    • Test case is closed
  4. Test report footer is written to the standard output
  5. Debug session ends in dead loop
