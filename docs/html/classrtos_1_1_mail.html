Detailed Description

template<typename T, uint32_t queue_sz>
class rtos::Mail< T, queue_sz >

The Mail class allow to control, send, receive, or wait for mail. A mail is a memory block that is send to a thread or interrupt service routine.

Template Parameters
Tdata type of a single message element.
queue_szmaximum number of messages in queue.
