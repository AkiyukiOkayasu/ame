Usbd_hidFunctions

USBD_User_HID_0.c implements the application specific functionality of the HID class and is used to receive and send data reports to the USB Host.

The implementation must match the configuration file USBD_Config_HID_0.h. The following values in USBD_Config_HID_0.h affect the user code:

  • 'Endpoint polling Interval' specifies the frequency of requests initiated by USB Host for USBD_HIDn_GetReport.
  • 'Number of Output Reports' configures the values for rid of USBD_HIDn_SetReport.
  • 'Number of Input Reports' configures the values for rid of USBD_HIDn_GetReport and USBD_HID_GetReportTrigger.
  • 'Maximum Input Report Size' specifies the maximum value for:
    • return of USBD_HIDn_GetReport
    • len of USBD_HID_GetReportTrigger.
  • 'Maximum Output Report Size' specifies the maximum value for len in USBD_HIDn_SetReport for rtype=HID_REPORT_OUTPUT
  • 'Maximum Feature Report Size' specifies the maximum value for len in USBD_HIDn_SetReport for rtype=HID_REPORT_FEATURE
