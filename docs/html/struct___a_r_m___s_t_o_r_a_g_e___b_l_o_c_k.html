◆ addr

uint64_t _ARM_STORAGE_BLOCK::addr

This is the start address of the storage block. It is expressed as an offset from the start of the storage map maintained by the owning storage controller.

◆ size

uint64_t _ARM_STORAGE_BLOCK::size

This is the size of the storage block, in units of bytes. Together with addr, it describes a range [addr, addr+size).
